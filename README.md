📚 Repositório de Estudos em Java
Este repositório reúne meus exercícios e práticas de Java, organizados por pastas.
A ideia é registrar minha evolução na linguagem e manter exemplos de conceitos importantes.
Estrutura
- DesafioControleFluxo/
Exercício sobre controle de fluxo, uso de exceções personalizadas e laços de repetição.
- SistemaReservaViagens/
Exercício proposto para modelagem de um sistema de reservas de viagens, com uso de classes, composição e enumerações.
- Biblioteca/
Exercício de modelagem de empréstimos de livros, com cálculo de valores de reposição.
- Restaurante/
Exercício de pedidos em restaurante, com cálculo de subtotal e total da conta.
- (novos exercícios serão adicionados aqui conforme avanço nos estudos)

Objetivo
- Consolidar fundamentos de POO (Programação Orientada a Objetos).
- Praticar conceitos como classes, métodos, enums, composição e exceções.
- Criar um histórico da minha evolução em Java.

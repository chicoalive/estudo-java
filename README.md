📚 Repositório de Estudos em Java
Este repositório reúne diversos exercícios e práticas de Java, organizados por pastas.
A ideia é registrar minha evolução na linguagem e manter exemplos de conceitos importantes.
Estrutura
- Cada pasta corresponde a um exercício ou prática independente.
- Os exercícios podem variar em complexidade: desde exemplos simples até projetos mais completos.
- Novos exercícios serão adicionados conforme avanço nos estudos.
Objetivo
- Consolidar fundamentos de Programação Orientada a Objetos (POO).
- Praticar conceitos como classes, métodos, enums, composição e exceções.
- Criar um histórico da minha evolução em Java.
